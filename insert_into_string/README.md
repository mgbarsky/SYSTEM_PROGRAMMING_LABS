# Insert into string
### Exercise 16

Given a string <tt>dest</tt>, the <tt>insert()</tt> function adds a copy of <tt>char c2</tt> after each instance of <tt>char c1</tt> in <tt>dest</tt>.

Write the body of the <tt>insert()</tt> functionin file src/insert.c.

You may assume that <tt>dest</tt> is guaranteed to have enough space to accommodate the newly inserted characters.

<hr><b>Sample Input</b><pre>char sample[7] = "momo";
insert(sample, 'm', 'o');
printf("%d", sample);
Output:
moomoo</pre>
